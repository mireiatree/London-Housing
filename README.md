# London-Housing
This analysis explores London housing prices to identify affordable areas.  Using the kaggle london housing, I analyze pricing patterns, value for money, and affordability across different neighborhoods.

# Skills
Python, pandas, matplotlib, seaborn

# Data Source
The dataset is available in this repository under housing.csv and was originally sourced from Kaggle. It contains housing price information across various London locations, including property types, prices, and postal codes.

# Data Cleaning
· Addressed missing location data (28% empty values) by utilizing postal codes to fill in gaps
· Converted 0 bedroom apartment to studios

# Analysis Techniques
· Bar charts to compare areas
· Histogram for property type and price distribution
· Analysed affordability

# Key Values
Best Value Areas
· Areas like Barwell Lane, Buckhurst Hill, and Walton-on-Thames offer the best value for properties under £600k, making them attractive options for budget-conscious buyers.

Property Type Pricing
· Penthouse is the most expensive property type, followed by standard house
· Bungalows and duplexes have very similar valuations
· Greenwich emerged as the cheapest area 

# Limitations
This analysis doesn't account for transport links
The dataset is 5 years old and may not reflect the current market
